vue项目部署到github的静态服务

2019-06-06
  • vuejs
  • github

本地写好vue项目以后。

  • 提交到远程仓库。
git add .
git commit -m 'dev'
git remote add origin https://github.com/xxx/xxx.git
git push origin master -f

  • 将docs下的所有文件夹推送到远程分支gh-pages
//打包
npm run build

//进入docs文件夹
cd ./docs

//提交到远程
git init
git add -A
git remote add origin https://github.com/xxx/xxx.git
git commit -m 'deploy'
git push -f git@github.com:xxx/xxx.git master:gh-pages
