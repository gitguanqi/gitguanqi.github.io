# 安装包

phpMyAdmin官网下载 (opens new window)

安装前，请使用xshell连接远程服务器。

这次使用wget进行下载安装。

  • 首先，进入下载文件夹。

这里是你存放下载包的地方。

cd /home/downloads
1
  • 其次，下载phpMyAdmin包。

如果没有wget安装可以下载yum install wget

wget https://files.phpmyadmin.net/phpMyAdmin/4.8.5/phpMyAdmin-4.8.5-all-languages.zip
1
  • 最后，解压安装包。

如果没有zip安装可以下载yum install zip unzip

unzip phpMyAdmin-4.8.5-all-languages.zip
1

# 基本设置

移动解压后的安装包到指定的文件夹中。

mv phpMyAdmin-4.8.5-all-languages /usr/share/nginx/html/myadm
cd /usr/share/nginx/html/myadm
1
2
  • 配置nginx站点
vi /etc/nginx/vhost/pam.conf
1
server {
  listen        80;
  server_name   phpmyadmin.guan.org;
  access_log    /var/log/nginx/phpmyadmin-access.log main;
  error_log     /var/log/nginx/phpmyadmin-error.log;
  location / {
    root    /usr/share/nginx/html/myadm;
    index   index.php;
  }
  location ~ \.php$ {
    root            /usr/share/nginx/html/myadm;
    fastcgi_pass    127.0.0.1:9000;
    fastcgi_index   index.php;
    fastcgi_param   SCRIPT_FILENAME /usr/share/nginx/html/myadm/$fastcgi_script_name;
    include         fastcgi_params;
  }
  location ~ /\.ht {
    deny all;
  }
}

# 网站预览

在本地hosts文件中加入一条dns记录。

192.168.1.123  phpmyadmin.example.org;
1

接下来在游览器输入phpmyadmin.example.org,就可以访问了。
