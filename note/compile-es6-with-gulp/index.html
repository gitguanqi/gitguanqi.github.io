导语：今天，来探讨一下gulp构建工具的使用。

# 安装依赖包

安装前你需要安装node.js和npm以及gulp三个包。

  • 首先，执行以下命令
npm init
1
  • 然后在安装 babel和babel-core两个包
npm install --save-dev gulp-babel babel-preset-env
npm install babel-cor --save-dev
1
2
  • 建立文件夹 新建一个名为gulpfile.js的文件夹,然后引入包
const gulp = require('gulp');
const babel = require('gulp-babel');

gulp.task('default', () =>
    gulp.src('es6/*.js')
        .pipe(babel({
            presets: ['env']
        }))
        .pipe(gulp.dest('dist'))
);
gulp.task('babel',() => {
gulp.watch('es6/*.js',['default'])
});
