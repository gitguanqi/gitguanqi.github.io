这篇文章主要介绍如何安装和使用一个插件市场的插件的方法。

# 目录

  • 查找插件
  • 安装插件
  • 使用插件

# 查找插件

打开插件市场：https://ext.dcloud.net.cn/ (opens new window)

例如我们项目中要用到富文本渲染的内容，可以查找一下富文本；

在结果页可以选一款适合自己的插件，我觉得mp-html这个插件挺好的；

mp-html 插件地址 (opens new window)

# 安装插件

点击右侧安装按钮下载插件并导入 HBuilderX；

下载完毕后你会发现你的根目录多出一个文件夹uni_modules，这个专门用于存放下载的插件。

这个文件夹下的插件可以直接使用，不需要引入全局或者局部组件。

# 使用插件

安装成功后开始使用插件。

  • 编写一段 js 变量并赋值；
data () {
    return {
        mdStr: `<h2>Hello</h2><p style="color: #f00;font-size: 20rpx;">我是富文本!</p>`,
    }
}
