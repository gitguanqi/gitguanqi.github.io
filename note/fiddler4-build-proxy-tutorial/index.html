最近在做本地代理调试，在网上找到一个好用的工具fiddler，使用以后发现非常好用，所以在此记录一下使用方法。使用代理可以避免自身ip地址被泄露，比较方便，安全。

# Fiddler简介

fiddler是一个http协议调试代理工具，支持网页网站页面调试，功能强大，易于扩展。

# 游览器设置

IE游览器：打开设置 -> internet选项 -> 连接 -> 局域网设置 -> 代理服务器高级 -> http地址填写127.0.0.1,端口写8888

# 案例

就以百度为例，在游览器打开百度的地址，会看到fiddler软件列表显示了代理的捕捉情况。

查看游览器网络请求就可以看到fiddler代理地址，不管请求哪个网站，都会是127.0.0.1:8888代理地址。
