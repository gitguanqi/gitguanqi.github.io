CentOS7系统更换为国内镜像

Mark   in HangZhou
2020-03-15
  • linux
  • mirror

最近在研究centos系统，由于国外的源太慢，所以想更换成国内的镜像，在此做一个记录！

# 国内常用yum镜像

# 企业

  • 搜狐开源镜像站: http://mirrors.sohu.com (opens new window)

  • 网易开源镜像站：http://mirrors.163.com (opens new window)

  • 阿里巴巴开源镜像站：https://developer.aliyun.com/mirror/ (opens new window)

# 大学

  • 北京理工大学: https://mirror.bit.edu.cn (opens new window)

  • 北京交通大学：https://mirror.bjtu.edu.cn (opens new window)

  • 兰州大学：http://mirror.lzu.edu.cn (opens new window)

  • 清华大学：https://mirrors.tuna.tsinghua.edu.cn (opens new window)

  • 中国科学技术大学：https://mirrors.ustc.edu.cn (opens new window)

  • 东北大学：http://mirror.neu.edu.cn (opens new window)

# 更新缓存

yum clean all
yum makecache
# 升级所有包、软件、系统内核
yum -y update
# 只升级所有包
yum -y upgrade
