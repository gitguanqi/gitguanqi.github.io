Linux中tcp端口检测

Mark   in HangZhou
2020-03-09
  • linux
  • tcping

在Linux系统中，有时候禁止ping还不行，有些人还可以通过ping tcp端口来获取你服务器上面开放的端口。今天就来说一下如何远程查看服务器上面的端口，是否开发及安全建议。

# 检查端口

# 安装工具

# window平台

安装地址 (opens new window)

点击下载，然后复制到电脑的system32目录下面。

# Linux平台

wget https://sources.voidlinux.eu/tcping-1.3.5/tcping-1.3.5.tar.gz
tar zxvf tcping-1.3.5.tar.gz
cd tcping-1.3.5/

2
3

如果没有安装gcc，安装一下：

yum install gcc
gcc -o tcping tcping.c
cp tcping /usr/bin

2
3

# 使用方法

# window平台的

  • 打开cmd命令行工具

  • 输入以下命令查看方法

tcping
1

  • 开始ping,这里以百度为例
tcping www.baidu.com 1234
tcping www.baidu.com 80
1
2

# linux平台的

还是百度为例。

tcping www.baidu.com 1234
tcping www.baidu.com 80
1
2

# 安全建议

如果你不想把服务器内部端口暴露出去，建议到云服务器厂商官网控制台设置安全组，只允许内网访问就可以禁止别人tcping了，即使ping到也会没有响应的。
