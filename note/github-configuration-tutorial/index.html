# зҷ»еҪ•GitHubзі»з»ҹ

зӮ№еҮ»еҸідёҠи§’иҙҰеҸ·еӨҙеғҸзҡ„вҖңв–јвҖқвҶ’SettingsвҶ’SSH kyesвҶ’Add SSH keyгҖӮ

# еӨҚеҲ¶id_rsa.pubзҡ„е…¬й’ҘеҶ…е®№

  • иҝӣе…Ҙc:/Users/xxxx_000/.ssh/зӣ®еҪ•дёӢпјҢжү“ејҖid_rsa.pubж–Үд»¶пјҢе…ЁйҖүеӨҚеҲ¶е…¬й’ҘеҶ…е®№гҖӮ
  • TitleиҮӘе®ҡд№үпјҢе°Ҷе…¬й’ҘзІҳиҙҙеҲ°GitHubдёӯAdd an SSH keyзҡ„keyиҫ“е…ҘжЎҶпјҢжңҖеҗҺвҖңAddKeyвҖқгҖӮ
